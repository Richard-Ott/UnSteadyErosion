Notes:

This requires input in version 3 format. Version 2 input format will not work here.

Information about the version 3 input format is here.

You can obtain calibration data already in version 3 input format at calibration.ice-d.org.

You can only do a production rate calibration for one nuclide at a time. Entering multiple-nuclide data will result in an error. Note that if you use one of the "calibration data set" pages available at calibration.ice-d.org, only one nuclide will be included. However, if you get data from a "site" page or a "publication" page, it may include measurements of more than one nuclide, in which case you would have to remove some of the data before entering it here.

This code is expected to be stable for exposure-age calculations for He-3 in pyroxene and olivine and Be-10, C-14, Ne-21, and Al-26 in quartz. Although it is also functional for He-3 in quartz, this pair is still under development and may change without warning.

If you encounter an error or a failure, please email me the data that you entered and a description of the problem. Thanks.
