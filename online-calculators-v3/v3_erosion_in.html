Information about the version 3 input format: Link

Note that the version 3 input format is the same for exposure age and erosion rate. That is, you have to enter a dummy value for the erosion rate, which is ignored. Version 2 has different input formats for age and erosion.

This code is expected to be stable for exposure-age calculations for He-3 in pyroxene and olivine and Be-10, C-14, Ne-21, and Al-26 in quartz. Although it is also functional for He-3 in quartz, this pair is still under development and may change without warning.

If you encounter an error or a failure, please email me the data that you entered and a description of the problem. Thanks.
